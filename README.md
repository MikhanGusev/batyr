
Building
--------

Dependencies (on Debian Wheezy/Ubuntu):

    sudo apt-get install libpoco-dev libpocofoundation9 libpoconet9 libpocoutil9 libgdal1 libgdal1-dev cmake g++ \
                build-essential libpq-dev


Building:

    cmake .
    make
